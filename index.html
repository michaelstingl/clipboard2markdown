Paste to Markdown

Instructions

  1. Find the text to convert to Markdown (e.g., in another browser tab)
  2. Copy it to the clipboard (Ctrl+C, or ⌘+C on Mac)
  3. Click a section button below or press 1, 2, 3 to paste with formatting
  4. Or paste directly (Ctrl+V) for plain append
  5. Clear the output (Ctrl+L, or ⌘+L on Mac)
  6. Download as .md file (Ctrl+S, or ⌘+S on Mac)

The conversion is carried out by Turndown, a Markdown converter written in JavaScript and running locally in the browser.
